# 资源文件夹说明

此文件夹包含所有预置的媒体资源，用于系统的资源库功能。

## 文件夹结构

```
resources/
├── docs/          # 教学文档
├── videos/        # 示范视频
├── ppt/           # PPT课件
├── cases/         # 案例分析
├── images/        # 图片素材
├── audio/         # 音频资源
└── uploaded/      # 用户上传的资源
```

## 如何添加预置资源

### 1. 教学文档 (docs/)
存放 PDF、Word、Excel 等文档文件：
- S-M量表评估指南
- WeeFIM评估手册
- 教案模板
- IEP指导文档

### 2. 示范视频 (videos/)
存放 MP4、AVI、MOV 等视频文件：
- 洗手七步法示范
- 刷牙技能示范
- 穿衣技能训练
- 如厕技能示范

### 3. PPT课件 (ppt/)
存放 PowerPoint 课件文件：
- 个人卫生习惯培养
- 饮食技能训练
- 社交技能课程

### 4. 案例分析 (cases/)
存放案例分析文档：
- 学生评估案例
- 成功训练案例
- 行为干预策略

### 5. 图片素材 (images/)
存放图片文件：
- 步骤图（PNG/JPG）
- 表情卡片
- 训练图片
- 图标素材

### 6. 音频资源 (audio/)
存放音频文件：
- 语音提示音
- 背景音乐
- 口令音频

## 文件命名建议

1. **使用有意义的名称**：避免使用特殊字符
2. **英文名称推荐**：避免中文命名可能导致的编码问题
3. **添加版本号**：如 `handwashing_v2.mp4`

## 注意事项

1. **文件大小限制**：
   - 单个文件建议不超过 100MB
   - 视频文件建议进行压缩

2. **文件格式**：
   - 文档：PDF（推荐）、DOC、DOCX、XLS、XLSX、PPT、PPTX
   - 视频：MP4（推荐）、AVI、MOV
   - 图片：PNG、JPG、GIF、BMP
   - 音频：MP3（推荐）、WAV、OGG

3. **版权声明**：
   - 确保所有预置资源拥有使用版权
   - 如有版权文件，请标注来源

## 构建和部署

运行构建命令时，这些资源会被自动复制到最终的 Electron 应用中：

```bash
npm run build:electron
```

## 开发环境

在开发环境下，资源文件会通过模拟路径访问。在 Electron 环境下，资源会保存到用户的 AppData 目录中。
