## 批量重命名命令（PowerShell）

```powershell
# 示例：将图片文件重命名为对应格式
# 假设您的原始图片名为 "感官手环.png"，运行以下命令：

Rename-Item "感官手环.png" "tactile-1.webp"
Rename-Item "感官石.png" "tactile-2.webp"
# ... 依此类推
```

## 图片规格提醒

- **推荐尺寸**: 96x96px 或 128x128px
- **推荐格式**: WebP（或 PNG）
- **文件大小**: 单张 < 20KB
- **存放位置**: `src/assets/images/equipment/`
