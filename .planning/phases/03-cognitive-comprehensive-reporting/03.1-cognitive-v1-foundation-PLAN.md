# Plan 03.1: Cognitive V1 Foundation

## Goal

Define a shippable V1 cognitive assessment contract so the repo has a concrete `cognitive` scale that can later plug into the existing `ScaleDriver` runtime.

## Context

- Phase 3 requires MOD-03, but there is no production cognitive scale in the repo yet.
- Historical product direction points to a teacher-facing cognitive screen around attention, memory, and executive function.
- The current architecture works best when question bank, interpretation rules, and persistence contract are fixed before driver/UI work starts.

## Requirements Covered

- MOD-03: User can conduct cognitive assessments using the ScaleDriver architecture.
