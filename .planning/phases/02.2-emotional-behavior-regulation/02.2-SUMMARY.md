## What Was Completed

1. Added a dedicated Emotional module route group under `/emotional/*`.
2. Added active emotional resource types and demo seed data for `emotion_scene` and `care_scene`.
3. Implemented a reusable session runtime in `useEmotionalSession.ts` with prompt escalation, retries, timing, and unified completion flow.
4. Implemented the `EmotionSceneTraining.vue` flow for scene observation, emotion recognition, reasoning, and response selection.
5. Implemented the `CareExpressionTraining.vue` flow for empathy / suggestion / action expression and receiver-perspective switching.
6. Added emotional training persistence, records page, trend report page, and `report_record(type='emotional')` integration.
7. Connected emotional resources into training plans so plan execution can launch both emotional training flows.
