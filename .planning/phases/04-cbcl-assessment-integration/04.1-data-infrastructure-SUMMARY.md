# Phase 04 Plan 04.1: CBCL Data Infrastructure Summary

**One-liner:** Established foundational data layer for CBCL assessment with comprehensive type definitions, 121-question bank, 6 norm groups with factor mappings, and expert feedback content.

## What Was Built

### 1. TypeScript Type Definitions (`src/types/cbcl.ts`)
Created comprehensive type system for CBCL assessment:
- **Base types:** `CBCLGender`, `CBCLAgeGroup`, `CBCLNormGroup` (6 combinations)
- **Question types:** `CBCLQuestion` with support for sub-items (56a-56h)
- **Norm types:** `CBCLFactorNorm` with items array, p69/p98 thresholds
- **Data types:** `CBCLSocialCompetenceData` for Part 2 form fields
- **Result types:** `CBCLScoreResult`, `CBCLFactorResult`, `CBCLSocialCompetenceResult`
- **Feedback types:** `CBCLFeedbackLevel`, `CBCLFactorFeedback`, `CBCLFeedbackConfig`
- **Function types:** Type definitions for helper functions

### 2. Question Bank (`src/database/cbcl-questions.ts`)
Created complete question database:
- **113 main behavior questions** (1-113) with 3-point Likert scale
- **8 sub-questions** (56a-56h) for somatic symptoms under question 56
- **Options configuration:** 0=无此表现, 1=有时有, 2=经常有
- **Helper functions:**
  - `getCBCLQuestions()` - Get all questions
  - `getCBCLMainQuestions()` - Get main questions only
  - `getCBCLSubQuestions()` - Get sub-questions only
  - `getCBCLQuestionById()` - Get specific question
  - `getCBCLOptionLabel()` - Get option text
  - `getCBCLQuestionCount()` - Get count statistics

### 3. Norm Configurations (`src/database/cbcl-norms.ts`)
Created comprehensive norm system:
- **Total score limits:** 6 norm groups with gender/age specific thresholds
- **Social norms:** Activity/social/school thresholds for 6-16 age groups
- **Factor norms:** Detailed factor mappings for 4 norm groups:
  - `factors_boy_6_11`: 9 factors (分裂样, 抑郁, 交往不良, 强迫性, 体诉, 社交退缩, 多动, 攻击性, 违纪)
  - `factors_girl_6_11`: 9 factors (体诉, 分裂样, 交往不良, 不成熟, 强迫性, 敌意性, 违纪, 攻击性, 多动)
  - `factors_boy_12_16`: 8 factors (焦虑强迫, 体诉, 分裂样, 抑郁退缩, 不成熟, 违纪, 攻击性, 残忍)
  - `factors_girl_12_16`: 9 factors (same as girl_6_11)
- **Helper functions:**
  - `determineNormGroup()` - Determine norm group by age
  - `getNormConfig()` - Get factor norms
  - `getSocialNorm()` - Get social competence norms
  - `getTotalScoreLimit()` - Get total score threshold
  - `evaluateFactorScore()` - Evaluate factor raw score

### 4. Feedback Configuration (`src/config/feedbackConfig.js`)
Added CBCL section to ASSESSMENT_LIBRARY:
- **Total score rules:** 3 levels (normal T<60, borderline 60-63, clinical 64+)
- **Social competence:** 3 levels with inverted interpretation (higher is better)
- **Syndrome scales:** 8 factor feedback configurations:
  - `anxious_depressed` - 焦虑与抑郁情绪
  - `withdrawn` - 社交退缩与孤立
  - `somatic` - 躯体化表现
  - `social_problems` - 社交互动障碍
  - `thought_problems` - 思维与认知的特殊性
  - `attention` - 注意力与执行功能
  - `rule_breaking` - 规则违抗与冲动行为
  - `aggressive` - 攻击与情绪爆发
- **Each level includes:** range, severity, title, content, structured_advice
- **Placeholder support:** `[儿童姓名]` for dynamic student name replacement
