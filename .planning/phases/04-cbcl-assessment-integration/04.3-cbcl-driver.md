# Plan 04.3: CBCL Driver & Scoring Engine

## Goal
Implement the CBCLDriver class extending BaseDriver with dynamic factor score calculation and comprehensive feedback generation.

## Context
CBCL scoring is complex:
- Two-part assessment: Social Competence (Part 2) + Behavior Problems (Part 3)
- Dynamic norm matching based on gender × age (6 groups)
- Different factor structures for each norm group
- T-score calculation for clinical interpretation
- Dual perspective feedback (social + behavior)

  answers: Record<string, ScaleAnswer>,
  context: StudentContext

  // 1. Store context
  this.studentName = context.name

    { code: 'school', name: '学校表现', rawScore: socialScores?.school.score || 0, ... },

    scaleCode: this.scaleCode,
    studentId: context.id,

    totalScore: behaviorScores.totalProblemsScore,
    level: behaviorScores.summaryLevel,

### Task 5: Implement generateFeedback Method
**File:** `src/strategies/assessment/CBCLDriver.ts`

Override abstract method:
```typescript
generateFeedback(scoreResult: CBCLScoreResult): CBCLStructuredFeedback {
  const feedbackConfig = ASSESSMENT_LIBRARY.cbcl

  // 1. Generate total scale feedback
  const totalTScore = scoreResult.behaviorProblems.totalProblemsTScore
  const totalLevelConfig = this.matchTScoreToLevel(
    feedbackConfig.total_scale.levels,
    totalTScore
  )

  // 2. Generate social competence feedback
  const socialFeedback = this.generateSocialCompetenceFeedback(
    scoreResult.socialCompetence,
    feedbackConfig.social_competence.levels
  )

  // 3. Generate syndrome scale feedback
  const syndromeFeedback = scoreResult.behaviorProblems.factors.map(factor => {
    const syndromeConfig = feedbackConfig.syndromes[factor.code]
    return {
      code: factor.code,
      name: factor.name,
      tScore: factor.tScore,
      level: factor.level,
      ...this.matchTScoreToLevel(syndromeConfig?.levels || [], factor.tScore)
    }
  })

  // 4. Replace placeholders
  const replacePlaceholders = (text: string) =>
    text.replace(/\[儿童姓名\]/g, this.studentName)

  // 5. Return structured feedback
  return {
    overallSummary: [replacePlaceholders(totalLevelConfig.summary)],
    overallAdvice: [replacePlaceholders(totalLevelConfig.expert_advice)],
    socialCompetence: {
      summary: replacePlaceholders(socialFeedback.summary),
      advice: replacePlaceholders(socialFeedback.expert_advice)
    },
    syndromeScales: syndromeFeedback.map(s => ({
      ...s,
      summary: replacePlaceholders(s.summary),
      advice: replacePlaceholders(s.expert_advice)
    })),
    clinicalProfile: {
      factors: scoreResult.behaviorProblems.factors,
      normGroup: scoreResult.behaviorProblems.normGroup
    }
  }
}
```

**Verification:** Feedback content matches reference engine with proper placeholder replacement.
