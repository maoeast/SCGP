# Phase 04 Plan 04.3: CBCL Driver & Scoring Engine Summary

## One-Liner
Implemented CBCLDriver class with complete scoring engine for Achenbach Child Behavior Checklist, supporting social competence calculation, dynamic behavior factor scoring for 6 norm groups, T-score conversion, and comprehensive feedback generation.

## What Was Built

### 1. CBCLDriver Core Structure (`src/strategies/assessment/CBCLDriver.ts`)
- Extended `BaseDriver` with CBCL-specific metadata
- Configured scale properties: code='cbcl', name='Achenbach儿童行为量表', version='家长版'
- Age range: 4-16 years (48-192 months), 113 questions
- Student context management (name, age, gender)
- Social data storage for Part 2 assessment

### 2. Social Competence Calculation
Implemented `calculateSocialCompetence()` method with 3 factors:
- **Activity Factor**: Sports (I) + Hobbies (II) + Organizations (III) + Labor (IV)
- **Social Factor**: Friends count (V) + Meetings (VI_a/b/c) + Peer relations (VI_d)
- **School Factor**: Academic performance (VII_1) + Special education flags (VII_2/3/4)
- Norm-based threshold comparison from `CBCL_SOCIAL_NORMS`
- Status determination: normal (0) / borderline (1) / clinical (2)

### 3. Behavior Problem Calculation
Implemented `calculateBehaviorProblems()` with dynamic factor mapping:
- **6 Norm Groups**: boy_4_5, boy_6_11, boy_12_16, girl_4_5, girl_6_11, girl_12_16
- **Factor Structures**: 8-9 factors per group (e.g., 抑郁, 攻击性, 多动, 体诉)
- **Raw Score Calculation**: Sum of item responses (0-2 scale)
- **T-Score Estimation**: Linear conversion using p69/p98 thresholds
- **Level Assignment**: normal (<65) / borderline (65-69) / clinical (>=70)
- **Broadband Scores**: Internalizing + Externalizing composites

### 4. Comprehensive Feedback Generation
Implemented `generateFeedback()` method:
- Total scale feedback from `ASSESSMENT_LIBRARY.cbcl.total_score_rules`
- Social competence feedback with level matching
- Syndrome scale feedback for all behavior factors
- Placeholder replacement (`[儿童姓名]` -> actual name)
- Structured advice extraction from feedback config

### 5. Driver Registration
Updated `src/strategies/assessment/index.ts`:
- Imported CBCLDriver
- Added 'cbcl' entry to driverRegistry
- Exported CBCLDriver for direct access

```typescript
factorActivity = scoreI + scoreII + scoreIII + scoreIV

// Behavior Problem Scoring
for (const factor of factorConfig) {
  rawScore = sum(answers[item].score for item in factor.items)
  tScore = 50 + 10 * ((rawScore - mean) / sd)
  level = tScore >= 70 ? 'clinical' : tScore >= 65 ? 'borderline' : 'normal'
}

// Broadband Scores
internalizing = sum(抑郁, 社交退缩/抑郁退缩, 体诉)

```
