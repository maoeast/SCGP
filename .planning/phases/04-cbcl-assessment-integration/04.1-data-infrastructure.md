# Plan 04.1: CBCL Data Infrastructure

## Goal
Establish the foundational data layer for CBCL assessment by creating the question bank, norm configurations, TypeScript types, and feedback content.

## Context
CBCL (Achenbach儿童行为量表) requires complex data structures:
- 113 behavior questions with 3-point Likert scale
- Gender×Age specific norm configurations (6 norm groups)
- Dynamic factor mappings (8-9 factors per group, different for each gender/age)
- Social competence scoring with 3 factors
- Clinical profile feedback content

## Tasks

### Task 1: Create CBCL TypeScript Types
**File:** `src/types/cbcl.ts`

Create comprehensive type definitions:
- `CBCLGender: 'boy' | 'girl'`
- `CBCLAgeGroup: '4_5' | '6_11' | '12_16'`
- `CBCLNormGroup: 'boy_4_5' | 'boy_6_11' | 'boy_12_16' | 'girl_4_5' | 'girl_6_11' | 'girl_12_16'`
- `CBCLQuestion` interface with support for sub-items (56a-56h)
- `CBCLFactorNorm` with items, p69, p98 thresholds
- `CBCLSocialCompetenceData` for Part 2 form data
- `CBCLBehaviorRawScores` and `CBCLFactorTScores` for results
- `CBCLScoreResult` extending base ScoreResult

**Verification:** Types compile without errors and cover all data structures from the research document.

### Task 2: Create CBCL Question Bank
**File:** `src/database/cbcl-questions.ts`

Create the 113 behavior questions array:
- Each question has id (1-113, plus 56a-56h), text, options [0,1,2]
- Include all questions from `docs/儿童行为量表（CBCL）- 2026数字化题目.md`
- Export `CBCL_QUESTIONS` array
- Export `CBCL_OPTIONS` for 3-point scale labels
- Create helper function `getCBCLQuestions(): ScaleQuestion[]`

**Verification:** All 113 questions + 8 sub-items (56a-h) are present with correct text.

### Task 3: Create CBCL Norm Configurations
**File:** `src/database/cbcl-norms.ts`

Create comprehensive norm configurations:
- `CBCL_TOTAL_SCORE_LIMITS` for each norm group
- `CBCL_SOCIAL_NORMS` for activity/social/school thresholds
- `CBCL_FACTOR_NORMS` object containing:
  - `factors_boy_6_11`: 9 factors (分裂样, 抑郁, 交往不良, etc.)
  - `factors_girl_6_11`: 9 factors (体诉, 分裂样, 交往不良, etc.)
  - `factors_boy_12_16`: 8 factors (焦虑强迫, 体诉, 分裂样, etc.)
  - `factors_girl_12_16`: 9 factors
  - Note: 4-5 age group has no detailed factor norms
- Each factor has items array (supports numbers and strings like '56a'), p69, p98
- Export `getNormConfig(gender, age)` helper function
- Export `determineNormGroup(gender, ageMonths)` function

**Verification:** All factor item mappings match the reference engine exactly.

### Task 4: Integrate CBCL Feedback Configuration
**File:** `src/config/feedbackConfig.js`

Add CBCL section to `ASSESSMENT_LIBRARY`:
- `cbcl.total_scale` with 3 levels (normal T<60, borderline 60-63, clinical 64+)
- `cbcl.social_competence` with 3 levels (T-score based, inverted interpretation)
- `cbcl.syndromes` with all factor-specific feedback:
  - anxious_depressed, withdrawn, somatic, social_problems
  - thought_problems, attention_problems, rule_breaking, aggressive
  - (factors vary by age/gender, use common superset)
- Each level has range_t_score, severity, title, summary, expert_advice
- Include placeholder `[儿童姓名]` for dynamic replacement

**Verification:** All feedback content is ported from `docs/儿童行为量表（CBCL）报告解释生成引擎.js`
