This is a gap-closure/polish phase to enhance the SDQ report page with expert feedback content from feedbackConfig.js. The deliverable is a fully functional report page that displays:

<decisions>
## Implementation Decisions

### Data Source
- Use `ASSESSMENT_LIBRARY.sdq` from `@/config/feedbackConfig.js`
- Total score rules: `ASSESSMENT_LIBRARY.sdq.total_score_rules[level]`
- Dimension rules: `ASSESSMENT_LIBRARY.sdq.dimensions[code][level]`

### Level Mapping
- Chinese level (`正常`, `边缘`, `异常`) maps to English level (`normal`, `borderline`, `abnormal`)
- This mapping already exists in SDQ_THRESHOLDS logic

### Template Structure
- Add "评估结果解读" card for total score content
- Enhance dimension table to show full content
- Replace hardcoded expertAdvice with dynamic advice from feedbackConfig

</decisions>
